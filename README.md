# Expense Tracker

**Автор:** [Шогенов Идар]

## Описание программы

Графическое приложение для учёта личных финансов. Позволяет добавлять расходы с указанием суммы и категории (еда, транспорт и т.д.), фильтровать их по категориям и датам, а также подсчитывать общие затраты за любой период. Все данные надёжно сохраняются в файл `expenses.json`.

## Как использовать

1. Запустите файл `expense_tracker.py`.
2. Введите сумму расхода в поле «Сумма».
3. Выберите категорию из выпадающего списка.
4. Укажите дату расхода (кликните по полю даты и выберите день в календаре).
5. Нажмите кнопку **«Добавить расход»**. Запись появится в таблице.
6. Используйте выпадающий список «Фильтр по категории» и кнопку **«Фильтровать»**, чтобы отобразить только нужные расходы.
7. Выберите даты начала и конца периода в полях календаря и нажмите **«Сумма за период»**, чтобы увидеть итоговую сумму.
8. Нажмите **«Сохранить данные»**, чтобы записать все расходы в файл.
9. При следующем запуске программа автоматически загрузит сохранённые данные.
10. Для работы календаря установите: pip install tkcalendar

## Примеры использования

*   **Сценарий:** Вы хотите узнать, сколько потратили на «Транспорт» в прошлом месяце.
    *   *Действие:* Выберите «Транспорт» в фильтре категорий и нажмите «Фильтровать».
*   **Сценарий:** Нужно посчитать общие расходы за неделю отпуска.
    *   *Действие:* В полях периода выберите даты начала и конца отпуска. Нажмите «Сумма за период».
*   **Тест:** Попробуйте ввести отрицательную сумму или буквы в поле «Сумма». Программа покажет сообщение об ошибке и не позволит сохранить некорректные данные.
